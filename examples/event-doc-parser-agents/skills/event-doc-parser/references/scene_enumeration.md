# 场景枚举定义

以下是系统支持的交通管控场景类型及其定义。Agent 在解析活动文档前，应先读取本文档，结合文档内容（前2000字符）判断场景类型。

| 场景代码 (scene_type) | 中文名称 | 典型特征关键词 | 备注 |
| :--- | :--- | :--- | :--- |
| `newyear` | 跨年/元旦 | 跨年夜, 元旦, 倒计时, 烟花秀 | |
| `marathon` | 马拉松 | 马拉松, 迷你跑, 半程马拉松, 赛道 | |
| `concert` | 演唱会 | 演唱会, 体育中心, 歌迷, 演出 | |
| `exhibition` | 博览会 | 博览会, 会展中心, 展馆, 交易会 | |
| `holiday` | 节假日 | 春节, 国庆, 五一, 假期出行 | |
| `other` | 其他 | (无法归类到上述场景) | 默认类型 |
