# 活动文档提取规则总结

本文档总结了活动文档提取的通用规则，包括时间处理、区域映射和动作映射。

---

## 提取规则总结

### 时间处理

| 原文表达 | 提取结果 |
|---------|---------|
| 15时起 | start_time: "2025-12-31 15:00" |
| 6时30分起 | start_time: "2025-03-23 06:30" |
| 至本区域比赛结束 | end_time: "本区域比赛结束" |
| 至比赛结束 | end_time: "比赛结束" |
| 视情恢复 | end_time: "视情恢复" |

### 区域类型映射

| 原文关键词 | type值 |
|-----------|--------|
| 核心管控区、核心区 | 核心区 |
| 疏导区、缓冲区 | 疏导区 |
| 分流区、外围区 | 分流区 |
| 比赛线路、赛道 | 线路 |
| XX大道、XX路 | 路段 |
| XX桥、高架桥 | 桥梁 |
| XX隧道、地下通道 | 隧道 |

### 动作类型映射

|---------|---------|

| 封闭 | 封闭 |
