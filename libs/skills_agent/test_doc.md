# 2026年跨年夜江北片区实施方案

## 一、活动概况
1. 时间：2026年12月31日18:00至2027年1月1日02:00
2. 地点：江北区核心商圈及滨江路段
3. 规模：预计参与人数20万人次

## 二、交通管控措施
### （一）核心管控区
范围：江北大道（建新南路至五红路）、北滨一路（金源路至五简路）合围区域。
措施：
1. 12月31日16:00起，核心区内禁止社会车辆停放。
2. 12月31日18:00起，核心区内禁止所有机动车通行（公交车除外）。

### （二）外围疏导区
范围：红黄路、建新北路、海尔路合围区域。
措施：
1. 视情采取分流措施，引导车辆绕行。

## 三、公共交通保障
1. 轨道交通：3号线、9号线延长运营时间至次日02:00。
2. 公交线路：增加夜间班次，加密发车频率。

## 四、应急处置
1. 设置3个医疗救助点。
2. 预留两条应急通道供特种车辆通行。
